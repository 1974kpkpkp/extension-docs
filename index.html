This is our new documentation section for extensions in Opera 14.

As Opera has switched from Presto to Chromium, our extensions infrastructure has also gotten a major overhaul: Opera's own OEX format is no longer supported, and in Opera 14, we've switched to Chromium's extension model. At this point, we support a subset of the Chromium extension APIs — with more to come —, as well as our own Speed Dial API.

To get you started, we've prepared a number of tutorials and have included relevant API documentation as well.

Let's get started! »
