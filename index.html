This is our new documentation section for extensions in Opera 14.

As Opera has switched from Presto to Chromium, our extensions infrastructure has also gotten a major overhaul: in Opera 14, Opera 11 & 12’s extension format is no longer supported, and instead, we've switched to Chromium’s extension model. At this point, Opera 14 supports a subset of the Chromium extension APIs — with more to come —, as well as our own Speed Dial API.

To get you started, we’ve prepared a number of tutorials and have included relevant API documentation as well. We’ve also created a conversion tool to make the transition as smooth as possible.

Let’s get started! »
