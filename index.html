This is our new documentation section explaining how to build extensions for the all new Chromium-based Opera browser for computers.

With Opera switching from Presto to Chromium and a complete UI remake, our extensions infrastructure has also gotten a major overhaul: from Opera 16 onward, Opera 11 & 12’s extension format is no longer supported, and instead, we've switched to Chromium’s extension model. At this point, Opera 16 supports a subset of the Chromium extension APIs — with more to come —, as well as our own Speed Dial API.

To get you started, we’ve prepared a number of tutorials and have included relevant API documentation as well. We’ve also created a conversion tool to make the transition as smooth as possible. If you have questions, contact us.
